> **Monitor de sistema en tiempo real para Linux** вҖ” CPU por nГәcleo, temperatura, frecuencia, RAM, SWAP, red con velocidad instantГЎnea, discos, procesos, grГЎficos histГіricos y salida JSON. Todo en terminal, sin dependencias pesadas.

---

## вңЁ CaracterГӯsticas

| MГіdulo | Detalle |
|---|---|
| **CPU** | Uso real por nГәcleo В· Frecuencia GHz В· Temperatura con colores |
| **Temperatura** | Soporte Intel (`coretemp`) В· AMD (`k10temp`) В· ACPI В· NVMe В· Auto-detecciГіn |
| **RAM / SWAP** | Uso en tiempo real con barras de progreso |
| **Red** | Velocidad вҶ“ / вҶ‘ instantГЎnea В· Total acumulado В· DetecciГіn automГЎtica de interfaz activa |
| **Discos** | Uso por particiГіn В· Bytes leГӯdos/escritos |
| **Procesos** | Top procesos ordenados por CPU% В· PID В· MEM% В· Usuario |
| **GrГЎficos histГіricos** | VisualizaciГіn ASCII del uso de CPU y RAM (Гәltimos 60 segundos) |
| **GPU** | Soporte NVIDIA (nvidia-smi), AMD (ROCm / sysfs), Intel (sysfs) |
| **Modo JSON** | Salida estructurada para scripts, i3blocks, polybar, cron |
| **UI** | Pantalla alternativa В· Cursor oculto В· Redimensionamiento dinГЎmico В· Salida limpia |

## рҹ“Ұ InstalaciГіn

### рҹ”ө Para todas las distribuciones (recomendado)

```bash
# Clonar el repositorio
git clone https://github.com/george0884/sysmonitorpro.git
cd sysmonitorpro

# Dar permisos al instalador
chmod +x install.sh

# Ejecutar instalador
./install.sh
```

El instalador:

В· Verifica/instala Python 3 y pip
В· Instala psutil (dependencia principal)
В· Pregunta por soporte opcional para GPU
В· Crea configuraciГіn en ~/.config/sysmonitorpro/config.json
В· Pregunta si deseas el comando global sysmonitor

рҹҹ  Debian / Ubuntu / Mint / Pop!_OS (manual)

```bash
sudo apt update
sudo apt install python3 python3-pip -y
pip3 install psutil
```

рҹ”ө Arch Linux / Manjaro / EndeavourOS (manual)

```bash
sudo pacman -S python python-psutil
```

рҹҹў Fedora / RHEL / AlmaLinux / Rocky Linux (manual)

```bash
sudo dnf install python3 python3-pip -y
pip3 install psutil
```

вҡҷпёҸ LГӯnea de comandos

OpciГіn DescripciГіn
--json Salida en formato JSON (para scripts)
--no-gpu Ocultar secciГіn GPU
--no-disks Ocultar secciГіn discos
--no-network Ocultar secciГіn red
--no-top Ocultar top procesos
-i N / --interval N Intervalo de actualizaciГіn en segundos (ej: -i 2.0)
-c ARCHIVO / --config ARCHIVO Usar archivo de configuraciГіn especГӯfico
--hist-size N TamaГұo del historial en segundos
--help Mostrar ayuda

Ejemplos

```bash
# Modo JSON para scripts
sysmonitor --json

# Ocultar GPU y discos
sysmonitor --no-gpu --no-disks

# Intervalo de 2 segundos
sysmonitor -i 2.0

# Usar configuraciГіn personalizada
sysmonitor -c ~/mi-config.json
```

вҡҷпёҸ ConfiguraciГіn personalizada

El script busca configuraciГіn en ~/.config/sysmonitorpro/config.json. Si no existe, usa valores por defecto.

Crear configuraciГіn manualmente

```bash
mkdir -p ~/.config/sysmonitorpro
nano ~/.config/sysmonitorpro/config.json
```

Opciones disponibles

```json
{
    "intervalo": 1.0,
    "mostrar_gpu": true,
    "mostrar_discos": true,
    "mostrar_red": true,
    "mostrar_top": true,
    "grafico_tamano": 40,
    "historial_segundos": 60,
    "interfaz_red": "auto"
}
```

OpciГіn Valores Defecto DescripciГіn
intervalo 0.5 - 5.0 1.0 Segundos entre actualizaciones
mostrar_gpu true / false true Mostrar u ocultar secciГіn GPU
mostrar_discos true / false true Mostrar u ocultar discos
mostrar_red true / false true Mostrar u ocultar red
mostrar_top true / false true Mostrar u ocultar top procesos
grafico_tamano 20 - 100 40 Ancho del grГЎfico histГіrico en caracteres
historial_segundos 30 - 300 60 DuraciГіn del historial en segundos
interfaz_red "auto" / Nombre "auto" Interfaz especГӯfica o detecciГіn automГЎtica

рҹ”Ң IntegraciГіn con otros programas

i3blocks

```ini
[sysmonitor]
command=sysmonitor --json | jq -r '"CPU: \(.cpu.percent)% RAM: \(.memory.ram.percent)%"'
interval=2
```

Polybar

```ini
[module/sysmonitor]
type = custom/script
exec = sysmonitor --json | jq -r '"CPU: \(.cpu.percent)%"'
interval = 2
```

Cron (registro cada minuto)

```bash
*/1 * * * * /usr/local/bin/sysmonitor --json >> /var/log/sysmonitor.log
```

tmux (en barra de estado)

```bash
# En ~/.tmux.conf
set -g status-right "#(sysmonitor --json | jq -r '\"CPU: \\(.cpu.percent)%\"')"
```

вқ“ SoluciГіn de problemas

Error: psutil not found

```bash
pip3 install --user psutil
# o en sistemas sin --user
pip3 install psutil --break-system-packages
```

No se ven temperaturas

```bash
sudo apt install lm-sensors
sudo sensors-detect
```

No se detecta GPU NVIDIA

```bash
nvidia-smi  # Debe mostrar informaciГіn
# Si no funciona, instalar drivers NVIDIA
```

El comando sysmonitor no se encuentra

```bash
# Reabrir la terminal o ejecutar:
source ~/.bashrc

# O agregar manualmente al PATH:
export PATH="$HOME/.local/bin:$PATH"
```
